# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-XX

### Added
- Initial release of HCP Packer Promote to Channel Action
- OIDC workload identity federation support for secure authentication
- Automatic organization and project ID detection via HCP CLI
- Channel promotion via HCP Packer API
- Comprehensive error handling and logging
- Support for custom HCP CLI versions
- Output variables: `success`, `channel_name`, `iteration_id`
- Example workflows for common use cases
- Complete documentation with Mermaid diagrams

### Features
- Secure authentication without long-lived credentials
- Automatic HCP CLI installation
- Detailed error messages for troubleshooting
- Support for custom workload identity provider headers
- Flexible credential file path configuration
